# 🔥 FIERY CALORIE

## 📌 Опис проєкту
**Fiery Calorie** — це вебдодаток для розрахунку добової норми калорій, індексу маси тіла (BMI) та макронутрієнтів (білки, жири, вуглеводи).

Проєкт розроблений як **курсова робота** студента спеціальності **«Комп’ютерна інженерія»**.

---

## 🌐 Тип проєкту
Це **вебдодаток**, який працює в браузері та не потребує встановлення.

---

## ⚙️ Основні можливості

- розрахунок **BMR (базовий обмін речовин)**  
- розрахунок **TDEE (добова потреба в калоріях)**  
- розрахунок **BMI (індекс маси тіла)**  
- вибір цілі:
  - схуднення  
  - підтримка форми  
  - набір маси  
- розрахунок **макронутрієнтів (білки, жири, вуглеводи)**  
- можливість задати **бажану вагу (повзунок)**  
- збереження **історії розрахунків (localStorage)**  
- перемикання мови **UA / ENG**
