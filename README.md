# FIERY_CALORIE

FIERY_CALORIE — це вебзастосунок для розрахунку добової норми калорій, базального обміну речовин (BMR), індексу маси тіла (BMI) та орієнтовного розподілу макронутрієнтів. Проєкт створений як навчальний застосунок з сучасним інтерфейсом, адаптивним дизайном і підтримкою світлої та темної теми.

## Можливості проєкту

- розрахунок BMR
- розрахунок BMI
- розрахунок добової норми калорій
- вибір цілі: схуднення, набір маси або підтримка форми
- використання бажаної маси тіла
- автоматичний перерахунок результатів
- відображення макронутрієнтів: білки, жири, вуглеводи
- збереження даних у localStorage
- історія розрахунків
- адаптивний інтерфейс
- світла та темна тема
